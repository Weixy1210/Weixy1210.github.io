SEO – Search English Optimization 搜索引擎优化

  1. TDK – title description keywords (权重递减)

  2. 语义化的html标签

  3. 提高搜索引擎排序结果的其他方法

    • 代码最上可以适当有<!- xxx ->注释

    • h1-h3标签使用起来，h1,h2标签一般一页只有一个

    • strong标签权重仅次于h1，em标签次于strong，b(加粗),u,i标签均有一定权重

    • 非装饰性图片加alt，增加网站的相关度

    • 对于重要的a链接，用title进行描述说明，el=”nofollow”阻止向外部链接爬取

    • 重要的html内容应该靠前

    • 重要内容不要通过js输出

    • 少用iframe，flash

    • 版权部分关键词强调

    • 提高网址速度

    • 首页、信息页面等制作成静态页面

    • 针对Google制作Sitemaps

    • 减少table嵌套，使用caption表格标题标签

    • 网站结构层次不能过多，一般三层

    • 使用面包屑导航

    • 谨慎使用display: none，对于不显示内容，设置z-index或设置到显示范围外
