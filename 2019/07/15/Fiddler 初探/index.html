Fiddler 初探

Posted on 2019-07-15 | In 整理积累

使用场景

拦截线上页面加载的文件，预调试修改内容

安装

官网下载地址

选择使用场景，输入邮箱，开始下载

双击安装包正常安装

配置 HTTPS 加密解析

1.Tool -> Fiddler Options -> HTTPS -> 勾选Decrypt HTTPS traffic

2.点击 Actions 按钮，选择 Trust Root Certificate

3.重启 Fiddler

使用

  • Ctrl + x 清除当前抓到的全部请求

  • 设置 Filters，有选择的抓取

    1.选择 Filters 选项卡，Request Headers 模块勾选 Show only if URL contains，输入自定义的 Filter 内容

    2.点击 Actions -> Run Filterset Now

  • 截取并修改请求文件

    点击底部栏切换至向下箭头暂停符，可以拦截服务器向浏览器发送的文件，修改后点击绿色 Run to Completion，发送给浏览器

  • 截取并替换请求文件
